# MiLensSimulator
# 小米水印模拟器

Generate Mi Lens watermark, including Leica and multiple logos

生成小米式徕卡水印，可选择不同品牌logo
<br><br>

1.下载fonts.zip并安装其中字体

2.将Leica.py和resource文件夹放在同一目录下

3.安装libs.txt中列出的库

4.运行Leica.py，选择要生成的文件和图标样式，点击Process
